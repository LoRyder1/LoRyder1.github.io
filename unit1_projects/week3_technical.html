Week #3:

Why is JavaScript popular in web development? What are the great things about JavaScript? What are it's limitations?

The reason Javascript is so popular in web development is because it is huge and has an empire and is still growing. It is kind of a self fufilling prophecy on popularity. Javascript is popular because it is an important part of the web and used the most widely; it is popular because it "is" popular. Another reason for JavaScript's popularity is its beginner-friendly nature.

Some of the great things about JavaScript is that its so easy. Beginners flow to JavaScript because its so easy. It is possible to creat something tangible in minutes. This is possible because the language naturally integrates with HTML the language of the world wide web.

Other benefits of JavaScript: works with any OS or computer, requires on complicated setup, requires no special software, it's free.

The limitation of Javascript involve not being able to multithread, cannot access web pages hosted on a different domain, cannot access databases, cannot write to files on the server, depends a lot on the browser, inaability to use local devices, option to disable JavaScript. Another big one : security issues.
